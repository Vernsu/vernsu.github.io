Swift 中的嵌套类型

发表于 2017-01-22

几种值类型和引用类型的嵌套结构

我们都知道值类型和引用类型的用法，日常使用时，经常会进行多层嵌套。这里，介绍下四种两级嵌套结构：

  • 值类型嵌套值类型
  • 值类型嵌套引用类型
  • 引用类型嵌套引用类型
  • 引用类型嵌套值类型

接下来分析时，使用 struct 代表值类型， class 代表引用类型。并且这里不考虑写时拷贝语义。

  • withUnsafePointer(to:_:) 函数可以用来打印值类型变量的内存地址。

  • print(Unmanaged.passUnretained(T).toOpaque())函数打印引用类型内存地址。
