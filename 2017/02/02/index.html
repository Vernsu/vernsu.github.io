使用协议扩展

将协议标记 @objc 意味着 protocol 只能被 calss 实现，struct 和 enum 无法使用。而且对于纯 swift 项目来说，引入 Objective-C 的东西相当让人不爽。

所以更加推荐的实现协议可选方法的方式是：使用 protocol extension。即我们利用协议的扩展实现我们想要可选的方法的默认实现。这样，这些方法在使用时就可以当成可选方法了。
